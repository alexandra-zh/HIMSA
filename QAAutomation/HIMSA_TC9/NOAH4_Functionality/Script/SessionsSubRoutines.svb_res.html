Name: IsDeviceInfoAdded(Ear, Field, Value)
Purpose: verifies whether the device was added to the devices list
Inputs: Field: the property of device that was specified in "Fit" module
examples: "Model", "Serial"
Ear: the ear, either "Right" or "Left"
Value: the data that was entered in the corresponding field in the "Fit" module
Return values: boolean, True if device was found in the devices list, False otherwise
