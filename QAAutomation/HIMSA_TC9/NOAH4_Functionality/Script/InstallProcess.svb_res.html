Name: InstallStart()
Purpose: runs setup.exe file and starts either first time installation or re-installation procedure 'Assumptions: path to setup.exe is assigned to SetupExePath in Main unit
Inputs: none
Return values: none


Name: CleanInstall()
Purpose: performs all necessary steps to install the application for the first time'Assumptions: selects "English (United States)" language by default
Inputs: none
Return values: none


Name: ReInstall()
Purpose: performs all necessary steps to re-install (repair) existing application
Inputs: none
Return values: none


Name: UninstallStart()
Purpose: runs setup.exe file and starts uninstallation procedure 'Assumptions: path to setup.exe is assigned to SetupExePath in Main unit; application is installed
Inputs: none
Return values: none


Name: UnInstall()
Purpose: performs all necessary steps to uninstall (remove) existing application
Inputs: none
Return values: none


Name: ContinueInstallation()
Purpose: continues first time installation after rebooting 'Assumptions: path to setup.exe is assigned to SetupExePath in Main unit
Inputs: none
Return values: none
